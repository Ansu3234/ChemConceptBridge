# Videos Directory

Place your video files here. They will be accessible via `/videos/filename.mp4` in the application.

## Required Video File

For the "Understanding pH and Acid–Base Reactions" module, place your video file here with the exact filename:

**Filename:** `Understanding pH and Acid–Base Reactions.mp4`

**Full path:** `frontend/public/videos/Understanding pH and Acid–Base Reactions.mp4`

## Notes

- The filename must match exactly, including the en-dash character (–) between "Acid" and "Base"
- Video files placed here will be served as static assets
- After adding the video file, restart your React development server if it's running
